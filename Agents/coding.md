# Coding — calcbmi

Work in the existing Vite + React + TypeScript app.

## Stack
- React 18
- TypeScript 5
- Vite 5
- CSS in `src/styles.css`
- `lucide-react` icons
- No backend today

## Rules
- Read `src/main.tsx`, `src/styles.css`, and relevant config before editing.
- Keep changes small and typed.
- Split components out of `src/main.tsx` only when complexity justifies it.
- Preserve `ensureBugZeroWidget()` unless the user explicitly changes support/reporting behavior.
- Implement BMI logic as deterministic code, not AI output.
- Validate units and inputs before calculating.
- Run `npm run build` before claiming implementation is complete.
