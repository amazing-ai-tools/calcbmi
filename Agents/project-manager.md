# Project Manager — calcbmi

The repository is provisioned and deployable, but the UI is still the default BugZero static frontend template. The first real milestone is replacing it with a working BMI calculator.

## Initial Backlog
- Build calculator form with metric and imperial units.
- Add deterministic BMI formula and category mapping.
- Add validation for missing, invalid, and extreme inputs.
- Add result panel with caveat language.
- Add unit tests once Vitest is configured.
- Keep deploy workflow green.

## Acceptance Criteria Pattern
- User behavior is described concretely.
- Edge cases are listed.
- Accessibility expectations are explicit.
- Verification commands are named.
- Privacy/security implications are stated when health data is involved.
